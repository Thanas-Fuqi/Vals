# Valentine Mini Websites

## Menu
The main page acts as a simple navigation hub for the project. It displays two
options that allow the visitor to navigate to one of the two interactive
Valentine-themed pages. The design is intentionally minimal so the focus stays
on choosing an experience rather than the menu itself.

## Over Loved
This page creates a playful animated experience where the user clicks to start
a loading sequence. A progress bar fills while hearts and visual effects appear
on the screen and a looping heartbeat sound plays in the background. The goal
is to create a fun feeling of being "over loved" using simple animation and
audio feedback.
[Go to Folder >>>](https://github.com/Thanas-Fuqi/Vals/tree/main/Over_Loved)

## Yes / No
This page asks the classic Valentine question: "Will you be my Valentine?" and
presents the user with Yes and No buttons. Choosing Yes reveals a message that
confirms the answer with a cute visual response, creating a lighthearted and
romantic interaction.
[Go to Folder >>>](https://github.com/Thanas-Fuqi/Vals/tree/main/Yes_No)
